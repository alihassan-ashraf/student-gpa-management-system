# 🎓 Student GPA Management System

A Python-based interactive web application developed using **Streamlit** to calculate a student's Semester Grade Point Average (SGPA) based on university grading criteria.

## 🚀 Live Demo

Try the application here:  
https://student-gpa-management-system-kjvmx5jtqqvco75ox9pqyk.streamlit.app/

---

## 📌 Project Overview

The Student GPA Management System simplifies semester GPA calculation. Users can enter student details, course names, credit hours, and marks, and the application automatically calculates grades, grade points, quality points, and final SGPA.

This project was developed as part of my Python learning journey while pursuing the **TEVTA Punjab Data Analytics Course**.

## 📖 GPA Calculation Formula

```text
Quality Point = Credit Hours × Grade Point

Semester GPA = Total Quality Points ÷ Total Credit Hours
```

