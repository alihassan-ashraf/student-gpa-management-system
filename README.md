# 🎓 Student GPA Management System

A Python-based interactive web application developed using **Streamlit** to calculate a student's Semester Grade Point Average (SGPA) based on university grading criteria. The application accepts course details, credit hours, and marks, automatically calculates grades, grade points, quality points, and displays the final SGPA in a structured result card.

---

## 📌 Project Overview

The Student GPA Management System is designed to simplify the process of calculating semester GPA according to university grading policies. Instead of manually calculating grade points and quality points, users can simply enter their course information, and the application performs all calculations automatically.

This project was developed as part of my Python learning journey while pursuing the **TEVTA Punjab Data Analytics Course (2026)**.

## 📖 GPA Calculation Formula

```
Quality Point = Credit Hours × Grade Point

Semester GPA = Total Quality Points ÷ Total Credit Hours
```

## ▶️ How to Run the Project

### 1. Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/student-gpa-management-system.git
```

### 2. Navigate to the project folder

```bash
cd student-gpa-management-system
```

### 3. Install the required package

```bash
pip install -r requirements.txt
```

### 4. Run the application

```bash
streamlit run GPA_Calculator.py
```
